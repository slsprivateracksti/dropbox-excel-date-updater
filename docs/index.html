Features

3 つのタブで完結する
日付入力の自動化

介護・医療施設・バックオフィスなど、Dropbox を使って Excel を管理している方のために設計しています。

TAB — 01
📝

単体実行

  • Dropbox 内の Excel をダイアログで選択
  • シートをドロップダウンから選択（自動一覧表示）
  • セル範囲表記に対応（例：H3:H20 / I2, H3:H20）
  • 除外セル指定で特定セルだけスキップ
  • 日付形式を自由設定（M/D / YYYY/MM/DD など）
  • 実行前に 3 択確認ダイアログ
  • Excel を開きながらセルを選択して自動取込（pywin32）
TAB — 02
⚡

一括実行

  • 複数施設を同じ日付でまとめて処理
  • 左リスト＋右詳細で施設をナビゲート
  • 戻る・進むで施設を順番に確認
  • 保存済みパターンから複数選択してリストへ追加
  • Dropbox フォルダスキャン＋ AND/OR 検索で追加
  • チェックで各施設の実行／除外を切り替え
TAB — 03
🗂

履歴・パターン管理

  • 設定をパターンとして保存・次回から呼び出し
  • パターンの複写・編集・別名保存
  • AND/OR 検索によるパターン絞り込み
  • 実行ログを自動記録（日時・ユーザー・施設・結果）
  • ログを CSV でエクスポート
  • パターンを JSON でエクスポート/インポート（PC 間移行）

Quick Start

ソース版クイックスタート

Python 環境をお持ちの方は、4 ステップで起動できます。
環境を用意したくない方は main.exe をそのままお使いください。

1
Python 3.11+ をインストール
python.org から Python 3.11 以上をインストール。「Add Python to PATH」にチェックを忘れずに。
2
ライブラリをインストール
ソース zip を展開後、コマンドプロンプトでツールフォルダへ移動し実行。
3
設定ファイルを作成
サンプルをコピーして、施設名・ファイルパス・シート名を編集。
4
起動
メインウィンドウが表示されたら準備完了。詳細は INSTALL.md を参照。
cmd / PowerShell 
# ライブラリをインストール
pip install -r requirements.txt

# 設定ファイルを作成
copy config.xlsx_date_targets.sample.json config.xlsx_date_targets.json

# config.xlsx_date_targets.json を
# テキストエディタで編集して施設情報を入力

# 起動
python main.py

Requirements

動作要件

Windows 専用ツールです。以下の環境をご確認ください。

🪟
OS
Windows 10 / 11
64bit 推奨。macOS・Linux は非対応。
🐍
Python
3.11 以上
ソース版のみ必要。main.exe は不要。
📦
Dropbox
デスクトップアプリ
%USERPROFILE%\Dropbox が存在すること。
📊
Excel ファイル
.xlsx / .xlsm
Dropbox 同期済みのファイルに対して動作します。
⚠ SmartScreen・Defender の警告について： main.exe はコード署名なしのため、Windows Defender SmartScreen が「保護されました」と表示する場合があります。 これは署名なし exe に対する通常の挙動です。「詳細情報」→「実行」で起動できます。 不安な場合はソース版（zip）をご利用ください。

Notes

注意事項

⚠ 設定ファイルの作成が必要です。 初回は config.xlsx_date_targets.sample.json を config.xlsx_date_targets.json にコピーし、施設名・ファイルパス・シート名を編集してください。このファイルは環境ごとに個別の設定が必要です。
⚠ 書き込み中は Excel ファイルを開かないでください。 ツールによる書き込み処理中に対象の Excel ファイルを別のアプリで開いていると、保存に失敗する場合があります。実行前に Excel を閉じてください。
⚠ Windows 専用ツールです。 本ツールは Windows 10 / 11 でのみ動作します。macOS・Linux には対応していません。
🛡 SmartScreen 警告について。 main.exe はコード署名がないため、初回起動時に Windows SmartScreen または Defender の警告が出る場合があります。「詳細情報」→「実行」で起動できます。不安な場合はソースコード版（zip）をご利用ください。
