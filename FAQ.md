# よくある質問 / FAQ

---

## 起動・セットアップ

### Q. 起動時に `ModuleNotFoundError: No module named 'openpyxl'` と表示される

依存ライブラリがインストールされていません。以下を実行してください。

```cmd
pip install -r requirements.txt
```

それでも解決しない場合は、Python が複数インストールされている可能性があります。  
`python -m pip install -r requirements.txt` を試してください。

---

### Q. 起動時に「Dropboxフォルダが見つかりません」と警告が出る

以下を順番に確認してください。

1. Dropbox デスクトップアプリがインストールされているか
2. Dropbox が起動・サインイン済みか
3. `C:\Users\ユーザー名\Dropbox` フォルダが存在するか

Dropbox フォルダを別の場所に移動している場合、ツール内の `get_dropbox_path()` 関数（`main.py` 冒頭）のパスを手動で変更してください。

```python
def get_dropbox_path() -> str:
    return r"D:\MyDropbox"  # ← 実際のパスに変更
```

---

### Q. `python main.py` を実行しても何も表示されない

以下を確認してください。

- コマンドプロンプトでエラーメッセージが出ていないか確認する
- `python --version` で Python 3.11 以上であることを確認する
- `tkinter` が使えるか確認する（`python -m tkinter` でテストウィンドウが開けば OK）

## セル選択・入力

### Q. 「Excelで選択」ボタンを押しても反応しない

`pywin32` がインストールされていません。

```cmd
pip install pywin32
```

インストール後も動作しない場合は、管理者権限のコマンドプロンプトで以下を実行してください。

```cmd
python Scripts/pywin32_postinstall.py -install
```

---

### Q. セルを範囲で指定したい（例：H3 から H20 まで）

対象セル・除外セル欄で `H3:H20` と入力してください。  
単独セルとの混在も可能です。

| 入力例 | 意味 |
|--------|------|
| `I2` | I 列 2 行目のみ |
| `H3:H20` | H 列 3〜20 行目（18 セル） |
| `I2, H3:H20, C5` | I2・H3〜H20・C5 の混在 |
| `B2:D4` | B〜D 列 × 2〜4 行の矩形（9 セル） |

---

### Q. 対象セルの中の一部だけスキップしたい

**除外セル**欄にスキップしたいセルを入力します。

**例：H3〜H20 のうち H10 と H15 だけ書き込まない場合**

| 項目 | 入力値 |
|------|--------|
| 対象セル | `H3:H20` |
| 除外セル | `H10, H15` |

→ H10 と H15 だけスキップされ、残りの 16 セルに書き込まれます。

## 日付・書式

### Q. 日付形式はどう設定すればよいか

`⚙設定` ボタンを押してダイアログを開き、書式記号を組み合わせて自由に設定できます。

| 書式記号 | 意味 | 例（2026年4月8日） |
|---------|------|-----------------|
| `YYYY` | 4 桁年 | `2026` |
| `MM` | 月（ゼロ埋め） | `04` |
| `M` | 月（ゼロなし） | `4` |
| `DD` | 日（ゼロ埋め） | `08` |
| `D` | 日（ゼロなし） | `8` |

**よく使う形式の例**

| 設定値 | 出力例 |
|--------|--------|
| `M/D` | `4/8` |
| `YYYY/MM/DD` | `2026/04/08` |
| `YYYY-MM-DD` | `2026-04-08` |
| `YYYY年MM月DD日` | `2026年04月08日` |
| `M月D日` | `4月8日` |

---

### Q. Excel に書き込んだ日付の表示がおかしい（数値や別の形式になる）

Excel のセルの書式設定が「文字列」になっている可能性があります。

1. 対象セルを選択
2. 右クリック →「セルの書式設定」
3. 「日付」または「ユーザー定義」を選択して OK

また、ツール側の日付形式設定が Excel の書式と合っているか確認してください。

## パターン・履歴

### Q. 設定を保存して次回から呼び出したい

① 単体実行タブで設定を入力後、「このパターンを保存」ボタンをクリックします。  
次回は ③ タブの「単体実行タブで開く」から呼び出せます。

---

### Q. 別の PC でも同じパターン設定を使いたい

③ 履歴・パターン管理タブの「設定エクスポート」で `patterns.json` を書き出し、  
別 PC で「設定インポート」で読み込んでください。

---

### Q. 誤って実行してしまった。元に戻せるか

**ツール側には Undo 機能はありません。**  
ただし、③ タブの実行ログで「いつ・誰が・どのセルに・何の日付を書き込んだか」を確認できます。  
Excel 側で正しい値に手動修正してください。

実行前に必ず確認ダイアログ（3択）が表示されるので、内容をよく確認してから「はい」を押してください。

---

### Q. パターンを複写して少し変えた設定を作りたい

2 通りの方法があります。

**方法A：「複写」ボタン**  
③ タブでパターンを選択 → 「複写」ボタン → `_コピー` という名前で複製されて編集ダイアログが開く

**方法B：「別名で保存」**  
③ タブでパターンを選択 → 「編集」ボタン → 内容を変更 → 「別名で保存」ボタン

---

## 一括実行

### Q. 一括実行で特定の施設だけ除外したい

② タブで対象施設を選択し、右側の「この施設を一括実行に含める」のチェックを外してください。  
チェックを外した施設は一括実行の対象外になります（リストからは削除されません）。

---

### Q. 「設定不足」という警告が出て一括実行できない

一括実行リストの中に、ファイルパスまたは対象セルが設定されていない施設があります。  
該当施設を選択して「← 戻る / 進む →」で確認し、編集または削除してください。

---

## ログ・エクスポート

### Q. 実行ログを Excel で確認したい

③ タブの「ログをCSVで保存」ボタンで CSV ファイルとして書き出し、Excel で開いてください。

---

### Q. ログはどこに保存されているか

ツールのフォルダ内の `execution_log.csv` に自動保存されます。  
このファイルは `.gitignore` で Git 管理対象外になっています。

---

### Q. ログを削除したい

`execution_log.csv` を直接削除してください。次回実行時に空のファイルが自動生成されます。

### Q. 対応している Excel ファイル形式は？

| 拡張子 | 対応状況 |
|--------|---------|
| `.xlsx` | ✅ 対応（推奨） |
| `.xlsm` | ✅ 対応（マクロ有効ブック） |
| `.xls` | ⚠ 読み取り専用（書き込み非推奨） |
| `.csv` | ❌ 非対応 |

`.xls` 形式は `openpyxl` が対応していないため、事前に `.xlsx` に変換してください。
